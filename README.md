# 🌦️ Sensor Dashboard for IoT Weather Station

## 📌 Overview
This repository contains the frontend dashboard for my **Remote Weather Station Project**. It is designed to provide real-time, continuous environmental monitoring by visualizing data collected from custom-built IoT hardware. 

*(Note: This version currently utilizes sample data to demonstrate the UI layout, data structuring, and visualization capabilities).*

## ✨ Key Features
* **Atmospheric Monitoring:** Visualizes precise readings for Temperature, Humidity, Pressure, and Altitude.
* **Gas & Air Quality Tracking:** Displays metrics from integrated gas sensors, including CO₂, Methane, and NH₃ levels.
* **Continuous Monitoring:** A clean, intuitive user interface designed to make complex environmental data easily readable at a glance.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Hardware Context:** Designed to eventually interface with custom IoT hardware (ESP32/Microcontrollers) via REST API/WebSockets.
